## Lina (100.40.1.209) OpenClaw YOLO Mode 設定

 Lina 在 OpenClaw 2026.4.1 升級後一直出現 "Exec approval is required" 錯誤。

### 解決方案：設定 YOLO Mode（不需要審批）

在 Lina 主機上執行：

1. 設定 host approvals 為 YOLO 模式
2. 設定 tools.exec 參數
3. 重啟 Gateway

### 設定內容
- tools.exec.host = gateway
- tools.exec.security = full
- tools.exec.ask = off
- exec-approvals.json defaults: security=full, ask=off, askFallback=full
